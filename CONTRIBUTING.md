## 🚀 如何贡献

1. **Fork 本仓库**
2. **创建特性分支**
   ```bash
   git checkout -b feature/AmazingFeature
   ```
3. **提交更改**
   ```bash
   git commit -m 'Add AmazingFeature'
   ```
4. **推送到分支**
   ```bash
   git push origin feature/AmazingFeature
   ```
5. **开启 Pull Request**

## 📝 贡献规范

### 知识文件格式
每个知识文件应包含以下字段：
- 基础信息（来源、分类、性味归经等）
- 功效主治
- 倪师讲解（倪海厦的讲解内容）
- 配伍应用/禁忌

### 提交规范
- 使用中文提交信息
- 按模块分类提交（如：`feat: 补全 100 个穴位`）
- 一次提交只做一个模块的修改

## 📞 联系方式

- GitHub Issues: 提交问题
- Discussion: 讨论交流
