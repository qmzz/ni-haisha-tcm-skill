# 小青龙汤

**来源：** 伤寒论  
**分类：** 太阳病方  
**功效：** 解表散寒，温肺化饮


## 组成

| 药物 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| **麻黄** | 9g | 君 | 发汗解表 |
| **桂枝** | 9g | 臣 | 助阳化饮 |
| **干姜** | 9g | 臣 | 温肺化饮 |
| **细辛** | 3g | 臣 | 温肺散寒 |
| **五味子** | 6g | 佐 | 敛肺止咳 |
| **白芍** | 9g | 佐 | 敛阴和营 |
| **半夏** | 9g | 佐 | 燥湿化痰 |
| **炙甘草** | 6g | 使 | 调和诸药 |


## 主治

### 太阳伤寒兼水饮内停

**主症：**
- 发热恶寒
- 无汗
- **咳喘**
- **痰多清稀**
- 脉浮

**病机：** 外寒内饮


## 用法

**煎服法：**
```
水煎服，温覆取汗。
```


## 🌿 倪师讲解

### 方义分析

**倪海厦：**
> "小青龙汤为外寒内饮主方。
> 麻黄桂枝解表，干姜细辛温肺化饮。
> 外寒内饮，非此方不能解。"

### 临床应用

**倪师经验：**
- 哮喘发作
- 慢性支气管炎
- 肺气肿


## 禁忌与注意事项

### 禁忌症

**倪师提醒：**
- ❌ 热咳禁用
- ❌ 阴虚干咳禁用
