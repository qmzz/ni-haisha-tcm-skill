# 阴阳辨证

**分类：** 八纲辨证  
**来源：** 倪海厦人纪系列 - 黄帝内经

---

## 📖 概述

阴阳辨证是八纲辨证的总纲，是一切辨证的最高层次。

> **《素问·阴阳应象大论》：** "善诊者，察色按脉，先别阴阳。"

---

## 🔍 阴证与阳证鉴别

### 阳证

| 症状 | 表现 |
|------|------|
| 寒热 | 发热，恶热 |
| 面色 | 红赤 |
| 声音 | 高亢有力 |
| 呼吸 | 气粗 |
| 口渴 | 渴喜冷饮 |
| 四肢 | 温热 |
| 小便 | 短赤 |
| 大便 | 秘结 |
| 舌象 | 舌红苔黄 |
| 脉象 | 浮数洪大 |

### 阴证

| 症状 | 表现 |
|------|------|
| 寒热 | 恶寒，肢冷 |
| 面色 | 苍白或晦暗 |
| 声音 | 低微无力 |
| 呼吸 | 气弱 |
| 口渴 | 口不渴或喜热饮 |
| 四肢 | 厥冷 |
| 小便 | 清长 |
| 大便 | 稀溏 |
| 舌象 | 舌淡苔白 |
| 脉象 | 沉迟细弱 |

---

## 🎯 阴阳辨证要点

### 一、辨阴阳盛衰

**阳盛：** 实热证  
**阴盛：** 实寒证  
**阳虚：** 虚寒证  
**阴虚：** 虚热证

### 二、辨阴阳互根

"阴生于阳，阳生于阴"
- 阳损及阴
- 阴损及阳
- 阴阳两虚

### 三、辨阴阳转化

- 重阳必阴
- 重阴必阳
- 寒极生热
- 热极生寒

---

## 📋 临床运用

### 阳证治法
- **治则：** 清热泻火，滋阴降阳
- **方剂举例：** 白虎汤、黄连解毒汤

### 阴证治法
- **治则：** 温阳散寒，补火助阳
- **方剂举例：** 四逆汤、理中丸

---

## ⚠️ 倪师提醒

> "阴阳辨证要抓主要矛盾。
> 有时候表面是热证，本质是寒证（真寒假热）。
> 有时候表面是寒证，本质是热证（真热假寒）。
> 一定要四诊合参，不能被假象迷惑。"
