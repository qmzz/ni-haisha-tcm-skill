# 三焦辨证

**所属分类：** 温病辨证  
**倪海厦讲解来源：** 《温病条辨》

---

## 📖 基础定义

### 什么是三焦辨证？

**三焦辨证**是温病（外感热病）的一种辨证方法，以**上、中、下三焦**为纲，概括温病发展过程中的病位和病机。

| 三焦 | 部位 | 脏腑 |
|------|------|------|
| 上焦 | 胸膈以上 | 心、肺 |
| 中焦 | 胸膈至脐 | 脾、胃、大肠 |
| 下焦 | 脐以下 | 肝、肾 |

---

## 🔥 上焦病证

### 病位
心、肺（包括心包）

### 证候类型

#### 1. 温邪犯肺
| 项目 | 内容 |
|------|------|
| 主症 | 发热、微恶风寒、咳嗽、头痛 |
| 舌脉 | 舌边尖红、苔薄白、脉浮数 |
| 治法 | 辛凉解表 |
| 代表方 | 银翘散 |

#### 2. 邪热壅肺
| 项目 | 内容 |
|------|------|
| 主症 | 身热、汗出、烦渴、咳嗽气喘 |
| 舌脉 | 舌红苔黄、脉数 |
| 治法 | 清热宣肺 |
| 代表方 | 麻杏石甘汤 |

#### 3. 邪陷心包
| 项目 | 内容 |
|------|------|
| 主症 | 神昏谵语、舌謇肢厥 |
| 舌脉 | 舌质红绛或纯绛、脉细数或滑数 |
| 治法 | 清心开窍 |
| 代表方 | 清宫汤、安宫牛黄丸 |

---

## 🔥 中焦病证

### 病位
脾、胃、大肠

### 证候类型

#### 1. 阳明热炽（胃经热盛）
| 项目 | 内容 |
|------|------|
| 主症 | 大热、大渴、大汗、脉洪大（四大症） |
| 舌脉 | 舌红苔黄燥 |
| 治法 | 清热生津 |
| 代表方 | 白虎汤 |

#### 2. 阳明腑实
| 项目 | 内容 |
|------|------|
| 主症 | 日晡潮热、大便秘结、腹满硬痛、谵语 |
| 舌脉 | 舌红苔黄厚燥、脉沉实有力 |
| 治法 | 峻下热结 |
| 代表方 | 大承气汤 |

#### 3. 湿热中阻
| 项目 | 内容 |
|------|------|
| 主症 | 身热不扬、脘痞腹胀、恶心呕吐 |
| 舌脉 | 舌苔黄腻、脉濡数 |
| 治法 | 辛开苦降 |
| 代表方 | 王氏连朴饮 |

---

## 🔥 下焦病证

### 病位
肝、肾

### 证候类型

#### 1. 肾阴耗损
| 项目 | 内容 |
|------|------|
| 主症 | 身热颧红、手足心热、口燥咽干 |
| 舌脉 | 舌红少苔、脉虚细 |
| 治法 | 滋阴补肾 |
| 代表方 | 加减复脉汤 |

#### 2. 虚风内动
| 项目 | 内容 |
|------|------|
| 主症 | 手足蠕动或瘛疭、神倦脉虚 |
| 舌脉 | 舌绛少苔、脉虚弱 |
| 治法 | 滋阴熄风 |
| 代表方 | 大定风珠 |

---

## 📊 三焦传变

### 顺传
```
上焦 → 中焦 → 下焦
温病由浅入深
```

### 逆传
```
肺 → 心包（逆传心包）
病情突然加重
```

---

## 💊 治疗原则

| 病位 | 治疗原则 |
|------|----------|
| 上焦 | 轻清宣透 |
| 中焦 | 辛开苦降 |
| 下焦 | 滋阴潜阳 |
