# 病因辨证

**所属分类：** 辨证方法  
**倪海厦讲解来源：** 《黄帝内经》

---

## 📖 基础定义

### 什么是病因辨证？

**病因辨证**是根据致病因素的来源、性质和特点，分析推断病证原因的辨证方法。

**分类：**
- 外感病因：六淫、疫疠
- 内伤病因：七情、饮食、劳逸
- 病理产物：痰饮、瘀血
- 其他：外伤、虫兽伤等

---

## 🌀 外感病因（六淫）

### 风邪
| 特性 | 表现 |
|------|------|
| 轻扬开泄 | 汗出、恶风 |
| 善行数变 | 游走性疼痛、动摇不定 |
| 为病 | 伤风、风湿、惊风 |

### 寒邪
| 特性 | 表现 |
|------|------|
| 寒为阴邪 | 畏寒肢冷 |
| 凝滞 | 疼痛固定 |
| 收引 | 拘急、无汗 |

### 暑邪
| 特性 | 表现 |
|------|------|
| 暑为阳邪 | 高热、心烦 |
| 耗气伤津 | 口渴、汗多、乏力 |
| 湿浊 | 身重、胸闷 |

### 湿邪
| 特性 | 表现 |
|------|------|
| 湿为阴邪 | 头身困重 |
| 粘滞 | 病程长、大便粘腻 |
| 重浊 | 下肢沉重、分泌物秽浊 |

### 燥邪
| 特性 | 表现 |
|------|------|
| 燥伤津液 | 口干、咽干、皮肤干 |
| 燥伤肺 | 干咳少痰、痰粘难咯 |

### 火（热）邪
| 特性 | 表现 |
|------|------|
| 火为阳邪 | 高热、面赤 |
| 炎上 | 头面部症状 |
| 伤津 | 口渴、便秘 |
| 生风动血 | 抽搐、出血 |

## 💓 内伤病因（七情）

| 七情 | 归脏 | 病机 |
|------|------|------|
| 喜伤心 | 心 | 喜则气缓 |
| 怒伤肝 | 肝 | 怒则气上 |
| 忧伤肺 | 肺 | 忧则气郁 |
| 思伤脾 | 脾 | 思则气结 |
| 悲伤肺 | 肺 | 悲则气消 |
| 恐伤肾 | 肾 | 恐则气下 |
| 惊伤肾 | 肾 | 惊则气乱 |

---

## 🍚 饮食劳逸

### 饮食所伤
| 类型 | 表现 |
|------|------|
| 饥饱失常 | 脘腹胀满、嗳腐吞酸 |
| 饮食不洁 | 吐泻、腹痛 |
| 偏嗜五味 | 营养失衡 |

### 劳逸失度
| 类型 | 表现 |
|------|------|
| 过度劳累 | 气虚乏力 |
| 过度安逸 | 气机不畅 |
