# 气血津液辨证

**所属分类：** 辨证方法  
**倪海厦讲解来源：** 《黄帝内经》

---

## 📖 基础定义

### 什么是气血津液辨证？

**气血津液辨证**是分析气、血、津液的生成、运行、功能异常，从而确定病证的辨证方法。

---

## 💨 气病辨证

### 1. 气虚证
| 项目 | 内容 |
|------|------|
| 病机 | 元气不足，脏腑功能减退 |
| 主症 | 神疲乏力、气短懒言、动则汗出 |
| 舌脉 | 舌淡苔白，脉虚 |
| 治法 | 补气 |
| 代表方 | 四君子汤 |

### 2. 气滞证
| 项目 | 内容 |
|------|------|
| 病机 | 气机郁滞，运行不畅 |
| 主症 | 胀痛、痞满、情绪抑郁 |
| 舌脉 | 舌淡红苔薄白，脉弦 |
| 治法 | 行气 |
| 代表方 | 柴胡疏肝散 |

### 3. 气逆证
| 项目 | 内容 |
|------|------|
| 病机 | 气机升降失常，应降不降 |
| 主症 | 咳喘、嗳气、呕吐、头痛眩晕 |
| 舌脉 | 舌红苔黄，脉弦数 |
| 治法 | 降气 |
| 代表方 | 苏子降气汤 |

### 4. 气陷证
| 项目 | 内容 |
|------|------|
| 病机 | 气虚升举无力 |
| 主症 | 脘腹坠胀、久泻脱肛、内脏下垂 |
| 舌脉 | 舌淡苔白，脉弱 |
| 治法 | 升提 |
| 代表方 | 补中益气汤 |

---

## 🩸 血病辨证

### 1. 血虚证
| 项目 | 内容 |
|------|------|
| 病机 | 血液不足，濡养功能减退 |
| 主症 | 面色苍白、头晕心悸、唇甲色淡 |
| 舌脉 | 舌淡苔白，脉细 |
| 治法 | 补血 |
| 代表方 | 四物汤 |

### 2. 血瘀证
| 项目 | 内容 |
|------|------|
| 病机 | 血液运行不畅，瘀滞停滞 |
| 主症 | 刺痛固定、肿块、紫绀 |
| 舌脉 | 舌有瘀斑，脉涩 |
| 治法 | 活血化瘀 |
| 代表方 | 血府逐瘀汤 |

### 3. 血热证
| 项目 | 内容 |
|------|------|
| 病机 | 热入血分，迫血妄行 |
| 主症 | 发热、出血、斑疹 |
| 舌脉 | 舌红绛，脉数 |
| 治法 | 清热凉血 |
| 代表方 | 犀角地黄汤 |

### 4. 血寒证
| 项目 | 内容 |
|------|------|
| 病机 | 寒凝血脉，运行不畅 |
| 主症 | 手足冷痛、肤色青紫 |
| 舌脉 | 舌淡紫，脉沉紧 |
| 治法 | 温经散寒 |
| 代表方 | 温经汤 |

---

## 💧 津液病辨证

### 1. 津液亏虚
| 项目 | 内容 |
|------|------|
| 病机 | 津液耗伤，滋润不足 |
| 主症 | 口干咽燥、皮肤干燥、尿少便干 |
| 舌脉 | 舌红少津，脉细数 |
| 治法 | 生津润燥 |
| 代表方 | 增液汤 |

### 2. 津液内停（水湿）
| 项目 | 内容 |
|------|------|
| 病机 | 脾失健运，水湿内停 |
| 主症 | 脘腹胀满、肢体浮肿 |
| 舌脉 | 舌淡苔白腻，脉濡 |
| 治法 | 利水渗湿 |
| 代表方 | 五苓散 |

### 3. 痰饮内停
| 项目 | 内容 |
|------|------|
| 病机 | 水湿凝聚为痰饮 |
| 主症 | 咳嗽痰多、胸闷脘痞 |
| 舌脉 | 舌苔腻，脉滑 |
| 治法 | 化痰蠲饮 |
| 代表方 | 二陈汤、苓桂术甘汤 |
