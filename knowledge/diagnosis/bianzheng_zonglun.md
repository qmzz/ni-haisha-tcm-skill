## 📊 中医辨证体系

中医辨证方法主要包括：

| 辨证方法 | 创立/传承 | 适用领域 |
|----------|-----------|----------|
| 八纲辨证 | 《黄帝内经》 | 基础辨证纲领 |
| 六经辨证 | 张仲景《伤寒论》 | 外感伤寒 |
| 卫气营血辨证 | 叶天士《温病条辨》 | 外感温病 |
| 三焦辨证 | 吴鞠通《温病条辨》 | 外感温病 |
| 脏腑辨证 | 《黄帝内经》 | 内伤杂病 |
| 经络辨证 | 《针灸甲乙经》 | 针灸治疗 |
| 气血津液辨证 | 《黄帝内经》 | 基础辨证 |
| 病因辨证 | 《黄帝内经》 | 病因分析 |

---

## 🎯 辨证选择指南

### 外感病
```
伤寒 → 六经辨证
温病 → 卫气营血辨证 + 三焦辨证
```

### 内伤病
```
脏腑为主 → 脏腑辨证
气血津液 → 气血津液辨证
经络相关 → 经络辨证
```

### 基础分析
```
定性 → 八纲辨证（阴阳表里寒热虚实）
定位 → 脏腑/经络/三焦
定因 → 病因辨证（风寒湿热痰瘀）
定量 → 气血津液辨证
```

## 🔄 辨证步骤（倪师心法）

### 第一步：定病位
- 表/半表半里/里
- 脏腑经络

### 第二步：定病性
- 寒/热
- 虚/实
- 阴/阳

### 第三步：定病势
- 传变方向
- 邪正关系

### 第四步：选治法
- 八法应用
- 方剂选择
