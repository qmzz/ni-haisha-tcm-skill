## 基本概念

**闻诊**是中医四诊之一，包括**听声音**和**嗅气味**两个方面，是通过听取患者的声音、呼吸、咳嗽、语言等声音变化，以及嗅察患者分泌物、排泄物的气味来诊断疾病的方法。

---

## 🌿 倪师讲解

> "闻诊者，听声音、嗅气味也。
> 声音者，肺之所主也；气味者，脾胃之所散也。
> 声音洪亮则气足，声音低微则气虚。
> 呼吸急促多属热，呼吸微弱多属寒。
> 口气酸臭则食积，口气腐臭则内痈。"

## 听声音

### 发声
| 声音 | 主病 |
|------|------|
| **嘶哑** | 实证：外感风寒或痰热；虚证：肺肾阴虚 |
| **失音** | 实证：外邪阻遏；虚证：精气亏虚 |
| **重听** | 实证：风邪上受；虚证：肾精不足 |

### 语言
| 语言 | 主病 |
|------|------|
| **谵语** | 实热证，热扰心神 |
| **郑声** | 虚证，心气大伤 |
| **狂言** | 痰火扰心 |
| **独语** | 心气不足，神失所养 |

### 呼吸
| 呼吸 | 主病 |
|------|------|
| **气粗** | 实证，热证 |
| **气微** | 虚证，寒证 |
| **喘** | 实证：痰热壅肺；虚证：肺肾气虚 |
| **哮** | 痰阻气道 |

### 咳嗽
| 咳嗽 | 主病 |
|------|------|
| **咳声重浊** | 风寒袭肺 |
| **咳声低微** | 肺气虚损 |
| **咳声不扬** | 热证 |
| **干咳** | 燥邪或阴虚 |

### 呕吐
| 呕吐 | 主病 |
|------|------|
| **呕吐清水** | 寒饮停胃 |
| **呕吐酸苦** | 热邪犯胃 |
| **喷射样呕吐** | 热扰心神 |

## 嗅气味

### 口气
| 气味 | 主病 |
|------|------|
| **酸臭** | 食积胃肠 |
| **腐臭** | 内痈（牙疳、咽喉溃烂） |
| **臭秽** | 胃热 |
| **腥臭** | 牙疳 |

### 分泌物
| 分泌物 | 气味 | 主病 |
|--------|------|------|
| **痰** | 腥臭 | 肺痈 |
| **涕** | 腥臭 | 鼻渊 |
| **带下** | 腥臭 | 湿热下注 |

### 排泄物
| 排泄物 | 气味 | 主病 |
|--------|------|------|
| **大便** | 酸臭 | 食积 |
| **大便** | 腥臭 | 寒湿 |
| **小便** | 臊臭 | 湿热下注 |
| **矢气** | 酸臭 | 食积 |

## 临床意义

### 声音与脏腑
- **声音出于肺** - 肺主气，声为气之旗
- **语言关乎心** - 心主神明，语言异常多责之心
- **脾胃主五臭** - 口气、分泌物之气味多与脾胃相关

### 闻诊与辨证
- **实证**：声音洪亮、气粗有力
- **虚证**：声音低微、气短无力
- **热证**：声音重浊、气味臭秽
- **寒证**：声音低微、气味腥冷
