## 基本概念

**十问歌**是中医问诊的纲领性口诀，概括了问诊的主要内容，使医者能够全面系统地收集病情资料。

---

## 🌿 倪师讲解

> "十问歌，问诊之纲领也。一问寒热二问汗，三问头身四问便，
> 五问饮食六问胸，七聋八渴俱当辨，九问旧病十问因，再兼服药参机变。
> 妇人尤必问经期，迟速闭崩皆可见，再添片语告儿科，天花麻疹全占验。"

---

## 十问歌原文

> 一问寒热二问汗，三问头身四问便，  
> 五问饮食六问胸，七聋八渴俱当辨，  
> 九问旧病十问因，再兼服药参机变，  
> 妇人尤必问经期，迟速闭崩皆可见，  
> 再添片语告儿科，天花麻疹全占验。

## 详解

### 一问寒热

| 寒热表现 | 辨证 |
|----------|------|
| **恶寒发热** | 外感表证 |
| **但寒不热** | 里寒证 |
| **但热不寒** | 里热证 |
| **寒热往来** | 少阳病 |

### 二问汗

| 汗出 | 辨证 |
|------|------|
| **有汗** | 表虚/里热 |
| **无汗** | 表实/津亏 |
| **自汗** | 气虚、阳虚 |
| **盗汗** | 阴虚 |
| **战汗** | 邪正相争 |
| **绝汗** | 阳气外脱 |

### 三问头身

| 症状 | 辨证 |
|------|------|
| **头痛** | 虚：气血不足；实：风火痰瘀 |
| **身痛** | 寒：痛如刺；湿：重着；虚：酸软 |
| **眩晕** | 风、痰、虚、火 |

### 四问便

| 大便 | 辨证 |
|------|------|
| **便秘** | 热结/阴虚/气虚 |
| **泄泻** | 寒湿/湿热/脾虚 |
| **便血** | 脾虚/湿热 |

| 小便 | 辨证 |
|------|------|
| **尿清长** | 虚寒 |
| **尿短黄** | 实热 |
| **尿频** | 肾虚/湿热 |
| **癃闭** | 实证：膀胱湿热；虚证：肾气虚 |

### 五问饮食

| 食欲 | 辨证 |
|------|------|
| **多食易饥** | 胃火/甲亢 |
| **食欲不振** | 脾虚/湿困 |
| **食后腹胀** | 脾虚/气滞 |
| **偏嗜异物** | 虫积 |

### 六问胸

| 症状 | 辨证 |
|------|------|
| **胸闷** | 气滞/痰阻 |
| **胸痛** | 心血瘀阻/痰热壅肺 |
| **心悸** | 心虚胆怯/心血不足 |
| **胁痛** | 肝郁/瘀血 |

### 七问耳聋

| 耳聋 | 辨证 |
|------|------|
| **突发耳聋** | 实证：肝胆火旺 |
| **渐发耳聋** | 虚证：肾精不足 |
| **耳鸣** | 实证：肝火；虚证：肾虚 |

### 八问渴

| 口渴 | 辨证 |
|------|------|
| **口渴喜冷** | 里热伤津 |
| **口渴不欲饮** | 湿热/瘀血 |
| **口干不渴** | 津液未伤 |
| **渴喜热饮** | 里寒证 |

### 九问旧病十问因

- **旧病**：既往病史、治疗经过
- **病因**：发病诱因、接触史

### 妇科问诊

| 问诊内容 | 意义 |
|----------|------|
| **月经周期** | 提前：血热；延后：血寒/血虚 |
| **经量** | 过多：血热/气虚；过少：血虚/寒凝 |
| **经色质** | 暗紫有块：血瘀；淡红：血虚 |
| **痛经** | 胀痛：气滞；刺痛：血瘀；隐痛：虚证 |

### 儿科问诊

| 问诊内容 | 意义 |
|----------|------|
| **出生情况** | 先天禀赋 |
| **喂养情况** | 后天营养 |
| **预防接种** | 传染病史 |
| **传染病史** | 天花、麻疹、水痘等 |
