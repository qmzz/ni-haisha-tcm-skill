# 气血失常

**所属分类：** 病机专题  
**倪海厦讲解来源：** 《黄帝内经》

---

## 📖 基础定义

### 什么是气血失常？

**气血失常**是指气的生成、运行、功能障碍，以及血液的生成、运行、功能异常，导致人体生理功能紊乱的病理状态。

---

## 🧬 气的失常

### 1. 气虚
| 项目 | 内容 |
|------|------|
| 病机 | 元气不足，脏腑功能减退 |
| 表现 | 神疲乏力、气短懒言、动则汗出、脉虚 |
| 治法 | 补气 |
| 代表方 | 四君子汤 |

### 2. 气滞
| 项目 | 内容 |
|------|------|
| 病机 | 气机郁滞，运行不畅 |
| 表现 | 胀痛、痞满、情绪抑郁、脉弦 |
| 治法 | 行气 |
| 代表方 | 柴胡疏肝散 |

### 3. 气逆
| 项目 | 内容 |
|------|------|
| 病机 | 气机升降失常，应降不降 |
| 表现 | 咳嗽、气喘、嗳气、呕吐、头痛眩晕 |
| 治法 | 降气 |
| 代表方 | 苏子降气汤 |

### 4. 气陷
| 项目 | 内容 |
|------|------|
| 病机 | 气虚升举无力，应升不升 |
| 表现 | 脘腹坠胀、久泻脱肛、内脏下垂、脉弱 |
| 治法 | 升提 |
| 代表方 | 补中益气汤 |

### 5. 气闭
| 项目 | 内容 |
|------|------|
| 病机 | 气机闭阻，外出严重障碍 |
| 表现 | 突然昏厥、四肢厥冷、牙关紧闭 |
| 治法 | 开窍 |
| 代表方 | 苏合香丸 |

### 6. 气脱
| 项目 | 内容 |
|------|------|
| 病机 | 气不内守，大量外泄 |
| 表现 | 面色苍白、汗出如油、呼吸微弱、脉微欲绝 |
| 治法 | 益气固脱 |
| 代表方 | 独参汤 |

---

## 🩸 血的失常

### 1. 血虚
| 项目 | 内容 |
|------|------|
| 病机 | 血液不足，濡养功能减退 |
| 表现 | 面色苍白、头晕心悸、唇甲色淡、舌淡脉细 |
| 治法 | 补血 |
| 代表方 | 四物汤 |

### 2. 血瘀
| 项目 | 内容 |
|------|------|
| 病机 | 血液运行不畅，瘀滞停滞 |
| 表现 | 刺痛、肿块、紫绀、舌有瘀斑、脉涩 |
| 治法 | 活血化瘀 |
| 代表方 | 血府逐瘀汤 |

### 3. 血热
| 项目 | 内容 |
|------|------|
| 病机 | 热入血分，迫血妄行 |
| 表现 | 发热、出血、斑疹、舌红绛、脉数 |
| 治法 | 清热凉血 |
| 代表方 | 犀角地黄汤 |

### 4. 血寒
| 项目 | 内容 |
|------|------|
| 病机 | 寒凝血脉，运行不畅 |
| 表现 | 手足冷痛、肤色青紫、遇寒加重、脉沉紧 |
| 治法 | 温经散寒 |
| 代表方 | 温经汤 |

---

## ⚔️ 气血同病

| 证型 | 病机 | 表现 | 治法 | 代表方 |
|------|------|------|------|--------|
| 气血两虚 | 气虚+血虚 | 气短乏力+血虚症状 | 气血双补 | 八珍汤 |
| 气滞血瘀 | 气滞+血瘀 | 胀痛+刺痛 | 行气活血 | 血府逐瘀汤 |
| 气不摄血 | 气虚+出血 | 乏力+各种出血 | 补气摄血 | 归脾汤 |
| 气随血脱 | 出血+气脱 | 大出血+气脱症状 | 益气固脱 | 独参汤 |
