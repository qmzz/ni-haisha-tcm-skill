# 津液代谢失常

**所属分类：** 病机专题  
**倪海厦讲解来源：** 《黄帝内经》

---

## 📖 基础定义

### 什么是津液代谢失常？

**津液代谢失常**是指人体津液的生成、输布、排泄功能障碍，导致津液亏虚或水湿内停的病理状态。

---

## 🧬 津液代谢过程

```
摄入（饮食、水液）
     ↓
脾胃运化（生成津液）
     ↓
肺宣发肃降（输布津液）
     ↓
三焦通调（运输通道）
     ↓
肾气化（排泄调节）
     ↓
膀胱贮存排出
```

**任一环节失常，皆可导致津液病变**

---

## 💧 津液不足（津亏）

### 1. 津伤
| 项目 | 内容 |
|------|------|
| 病机 | 津液轻度耗伤 |
| 表现 | 口干咽燥、皮肤干燥、尿少便干、舌红少津 |
| 治法 | 生津润燥 |
| 代表方 | 增液汤 |

### 2. 液脱
| 项目 | 内容 |
|------|------|
| 病机 | 津液严重耗伤，液质亏损 |
| 表现 | 皮肤皱褶、眼窝凹陷、骨瘦如柴、舌红无苔 |
| 治法 | 滋阴养液 |
| 代表方 | 大补阴丸 |

---

## 💦 水湿内停

### 1. 湿浊内生
| 项目 | 内容 |
|------|------|
| 病机 | 脾失健运，水湿内生 |
| 表现 | 脘腹胀满、纳呆、便溏、肢体困重、舌苔腻 |
| 治法 | 燥湿健脾 |
| 代表方 | 平胃散 |

### 2. 痰饮内停
| 项目 | 内容 |
|------|------|
| 病机 | 水湿凝聚为痰饮 |
| 表现 | 咳嗽痰多、胸闷脘痞、呕吐清水（见 [[痰饮]]） |
| 治法 | 化痰蠲饮 |
| 代表方 | 二陈汤、苓桂术甘汤 |

### 3. 水肿
| 项目 | 内容 |
|------|------|
| 病机 | 津液泛溢肌肤 |
| 表现 | 头面、眼睑、四肢、腹背浮肿 |
| 治法 | 利水消肿 |
| 代表方 | 五苓散、猪苓汤 |

---

## 🔍 辨证要点

### 津亏 vs 湿盛
| 鉴别点 | 津亏 | 湿盛 |
|--------|------|------|
| 舌象 | 舌红少津 | 舌淡苔腻 |
| 脉象 | 细数 | 濡缓 |
| 口渴 | 口渴欲饮 | 口不渴 |
| 小便 | 短赤 | 清长 |
| 皮肤 | 干燥 | 浮肿 |

---

## 💊 治疗原则

### 津亏
- **生津** - 用滋阴润燥药
- **补液** - 必要时静脉补液
- **忌汗吐下** - 防止津液进一步耗伤

### 水湿
- **燥湿** - 苦温燥湿
- **利湿** - 利水渗湿
- **逐水** - 攻逐水饮（用于实证）
- **温阳** - 温阳化气（用于阳虚水停）
