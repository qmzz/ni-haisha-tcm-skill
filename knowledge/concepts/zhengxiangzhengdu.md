# 正邪相争

**所属分类：** 病机专题  
**倪海厦讲解来源：** 《伤寒论》

---

## 📖 基础定义

### 什么是正邪相争？

**正邪相争**是中医病机学的核心概念，指人体正气与致病邪气之间的斗争状态。

| 要素 | 定义 |
|------|------|
| **正气** | 人体的抗病能力和生命活动功能 |
| **邪气** | 致病因素的总称（六淫、七情等） |

**经典：** 《素问·评热病论》"邪之所凑，其气必虚"

---

## 🧬 正邪相争的层次

### 1. 卫气营血层次
```
卫分 → 气分 → 营分 → 血分
正邪相争逐层深入
```

### 2. 六经层次
```
太阳 → 阳明 → 少阳 → 太阴 → 少阴 → 厥阴
正邪相争依次传变
```

### 3. 脏腑层次
病邪深入脏腑，与脏腑正气相争

## ⚔️ 正邪盛衰与疾病转归

### 邪胜正退（病进）
| 状态 | 表现 |
|------|------|
| 邪气盛 | 高热、剧痛、烦躁 |
| 正气退 | 神疲、乏力、脉弱 |
| 病机 | 邪气压制正气，病情加重 |

### 正胜邪退（病退）
| 状态 | 表现 |
|------|------|
| 正气复 | 热退、痛减、脉缓 |
| 邪气退 | 症状减轻，神清气爽 |
| 病机 | 正气驱邪外出，病情好转 |

### 正邪相持（病慢性化）
| 状态 | 表现 |
|------|------|
| 正虚邪恋 | 症状迁延不愈 |
| 病机 | 正气不能完全驱邪，邪气未能深入 |

### 邪盛正衰（病危）
| 状态 | 表现 |
|------|------|
| 邪气独盛 | 高热神昏、脉微欲绝 |
| 正气衰败 | 四肢厥冷、汗出如油 |
| 病机 | 阳气外脱，危及生命 |

## 🔍 六经病中的正邪相争

### 太阳病
- **正邪相争于表**
- 脉浮：正气向外抗邪
- 汗出：营卫失调

### 阳明病
- **正邪相争于里（胃肠）**
- 大热、大汗、大渴：正邪交争激烈

### 少阳病
- **正邪相争于半表半里**
- 往来寒热：正邪互有胜负

### 太阴病
- **脾胃阳虚**
- 邪从寒化，正气不足

### 少阴病
- **心肾阳虚**
- 邪从寒化，正气衰微

### 厥阴病
- **正邪最后相争**
- 寒热错杂，胜负未定

## 💊 治疗原则

### 扶正祛邪
| 方法 | 适应证 |
|------|--------|
| 扶正为主 | 正气虚为主 |
| 祛邪为主 | 邪气盛为主 |
| 攻补兼施 | 虚实夹杂 |

### 倪师治疗思路
1. **判断正邪形势** - 是攻是补
2. **因势利导** - 顺势驱邪外出
3. **顾护正气** - 祛邪不伤正
