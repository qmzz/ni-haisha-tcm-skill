# 五行

**分类：** 中医基础理论  
**来源：** 黄帝内经、难经

---

## 📖 核心概念

### 五行的定义

五行是指木、火、土、金、水五种物质及其运动变化。

> **《尚书·洪范》：** "五行：一曰水，二曰火，三曰木，四曰金，五曰土。"

### 五行的特性

| 五行 | 特性 | 引申义 |
|------|------|--------|
| **木** | 曲直 | 生长、升发、条达 |
| **火** | 炎上 | 温热、上升、光明 |
| **土** | 稼穑 | 生化、承载、受纳 |
| **金** | 从革 | 清洁、肃降、收敛 |
| **水** | 润下 | 寒凉、滋润、下行 |

## 🔄 五行生克

### 相生关系

```
木 → 火 → 土 → 金 → 水 → 木（循环）
```

| 相生 | 关系 | 比喻 |
|------|------|------|
| 木生火 | 母→子 | 木燃烧生火 |
| 火生土 | 母→子 | 火焚木成灰 |
| 土生金 | 母→子 | 土中藏金 |
| 金生水 | 母→子 | 金熔成液 |
| 水生木 | 母→子 | 水滋养木 |

### 相克关系

```
木 → 土 → 水 → 火 → 金 → 木（循环）
```

| 相克 | 关系 | 比喻 |
|------|------|------|
| 木克土 | 制约 | 树根破土 |
| 土克水 | 制约 | 土堤防水 |
| 水克火 | 制约 | 水灭火 |
| 火克金 | 制约 | 火熔金 |
| 金克木 | 制约 | 金斧伐木 |

## 🩺 临床应用

### 五行与疾病传变

#### 相生传变
- **母病及子：** 如肝病传心（木生火）
- **子病及母：** 如心病传肝（火→木）

#### 相克传变
- **相乘：** 相克太过，如肝气乘脾（木克土）
- **相侮：** 反克，如肝火犯肺（木侮金）

### 五行与治法

| 治法 | 原理 | 应用 |
|------|------|------|
| **培土生金** | 补脾益肺（土生金） | 肺虚久咳 |
| **滋水涵木** | 补肾养肝（水生木） | 肝肾阴虚 |
| **益火补土** | 温肾健脾（火生土） | 脾肾阳虚 |
| **金水相生** | 补肺滋肾（金生水） | 肺肾阴虚 |
| **抑木扶土** | 疏肝健脾（制木护土） | 肝脾不和 |

## 🌿 倪师讲解

### 五行与六经

倪海厦：**"五行是脏腑的基础，六经是五行的临床应用。"**

- **太阳病** - 主表，与膀胱经相关（水）
- **阳明病** - 主里热，与胃大肠相关（土、金）
- **少阳病** - 主半表半里，与胆经相关（木）
- **太阴病** - 主脾虚寒湿（土）
- **少阴病** - 主心肾阴阳（火、水）
- **厥阴病** - 主肝经寒热错杂（木）

### 五行与方剂

| 方剂 | 五行关系 | 应用 |
|------|----------|------|
| 小柴胡汤 | 疏肝利胆（木） | 少阳证 |
| 理中汤 | 温中健脾（土） | 太阴病 |
| 四逆汤 | 温肾回阳（水、火） | 少阴寒化 |
| 白虎汤 | 清肺胃热（金、土） | 阳明经证 |

## 📚 经典原文

### 《黄帝内经》

> **"五行者，金木水火土也。更贵更贱，以知死生，以决成败，而定五脏之气，间甚之时，死生之期也。"**  
> ——《素问·脏气法时论》

> **"木得金而伐，火得水而灭，土得木而达，金得火而缺，水得土而绝。"**  
> ——《素问·宝命全形论》
