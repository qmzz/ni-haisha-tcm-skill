# 经络

**分类：** 中医基础理论  
**来源：** 黄帝内经、难经

---

## 📖 核心概念

### 经络的定义

经络是运行气血、联络脏腑肢节、沟通上下内外的通路系统。

> **《灵枢·本脏》：** "经脉者，所以行血气而营阴阳，濡筋骨，利关节者也。"

### 经络系统组成

```
经络系统
├── 经脉（主干）
│   ├── 十二正经
│   ├── 奇经八脉
│   └── 十二经别
├── 络脉（分支）
│   ├── 十五别络
│   ├── 孙络
│   └── 浮络
└── 连属部分
    ├── 十二经筋
    └── 十二皮部
```

## 🩺 临床应用

### 经络辨证

| 症状 | 所属经络 | 取穴 |
|------|----------|------|
| 头痛项强 | 太阳经 | 风池、天柱 |
| 前额痛 | 阳明经 | 合谷、内庭 |
| 侧头痛 | 少阳经 | 外关、足临泣 |
| 巅顶痛 | 厥阴经 | 太冲、百会 |

### 循经取穴

- **本经取穴：** 某经病变取本经穴位
- **表里经取穴：** 取相表里经的穴位
- **同名经取穴：** 取手足同名经的穴位

## 🌿 倪师讲解

### 经络与六经

倪海厦：**"伤寒论的六经就是十二经脉的病变表现。"**

- **太阳经** - 足太阳膀胱经、手太阳小肠经
- **阳明经** - 足阳明胃经、手阳明大肠经
- **少阳经** - 足少阳胆经、手少阳三焦经
- **太阴经** - 足太阴脾经、手太阴肺经
- **少阴经** - 足少阴肾经、手少阴心经
- **厥阴经** - 足厥阴肝经、手厥阴心包经

### 经络与针灸

| 经脉 | 代表穴位 | 主治 |
|------|----------|------|
| 肺经 | 太渊、列缺 | 咳嗽、气喘 |
| 大肠经 | 合谷、曲池 | 头痛、牙痛 |
| 胃经 | 足三里、内庭 | 胃痛、呕吐 |
| 脾经 | 三阴交、血海 | 妇科病、血证 |
| 心经 | 神门、少府 | 心悸、失眠 |
| 小肠经 | 后溪、听宫 | 耳鸣、耳聋 |
| 膀胱经 | 委中、昆仑 | 腰痛、头痛 |
| 肾经 | 太溪、涌泉 | 肾虚、耳鸣 |
| 心包经 | 内关、劳宫 | 心痛、胃痛 |

| 胆经 | 阳陵泉、风池 | 胁痛、头痛 |
