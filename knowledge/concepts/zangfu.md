# 脏腑

**分类：** 中医基础理论  
**来源：** 黄帝内经、难经

---

## 📖 核心概念

### 脏腑的定义

脏腑是人体内脏的总称，分为五脏和六腑。

> **《素问·五脏别论》：** "所谓五脏者，藏精气而不泻也，故满而不能实。六腑者，传化物而不藏，故实而不能满也。"

### 五脏六腑

| 分类 | 器官 | 功能特点 |
|------|------|----------|
| **五脏** | 心、肝、脾、肺、肾 | 藏精气，满而不实 |
| **六腑** | 胆、胃、小肠、大肠、膀胱、三焦 | 传化物，实而不满 |

## 🫀 五脏功能

### 心（君主之官）

| 功能 | 说明 |
|------|------|
| **主血脉** | 推动血液运行 |
| **主神明** | 主宰精神意识思维 |
| **在体合脉** | 心与脉相连 |
| **在窍为舌** | 舌为心之苗 |
| **在志为喜** | 喜则气缓 |
| **在液为汗** | 汗为心之液 |

### 肝（将军之官）

| 功能 | 说明 |
|------|------|
| **主疏泄** | 调畅气机，促进消化 |
| **主藏血** | 贮藏血液，调节血量 |
| **在体合筋** | 筋赖肝血濡养 |
| **在窍为目** | 目为肝之窍 |
| **在志为怒** | 怒则气上 |
| **在液为泪** | 泪为肝之液 |

### 脾（仓廪之官）

| 功能 | 说明 |
|------|------|
| **主运化** | 运化水谷精微 |
| **主统血** | 统摄血液在脉内运行 |
| **主升清** | 升举清气 |
| **在体合肉** | 肌肉赖脾运化营养 |
| **在窍为口** | 口为脾之窍 |
| **在志为思** | 思则气结 |
| **在液为涎** | 涎为脾之液 |

### 肺（相傅之官）

| 功能 | 说明 |
|------|------|
| **主气司呼吸** | 主一身之气和呼吸之气 |
| **主宣发肃降** | 宣发卫气，肃降浊气 |
| **主通调水道** | 调节水液代谢 |
| **在体合皮** | 肺主皮毛 |
| **在窍为鼻** | 鼻为肺之窍 |
| **在志为悲** | 悲则气消 |
| **在液为涕** | 涕为肺之液 |

### 肾（作强之官）

| 功能 | 说明 |
|------|------|
| **藏精** | 贮藏先天和后天之精 |
| **主水** | 调节水液代谢 |
| **主纳气** | 摄纳肺吸入的清气 |
| **在体合骨** | 肾主骨生髓 |
| **在窍为耳** | 耳为肾之窍 |
| **在志为恐** | 恐则气下 |
| **在液为唾** | 唾为肾之液 |

## 🏺 六腑功能

### 胆（中正之官）
- **贮藏和排泄胆汁**
- **主决断**

### 胃（仓廪之官）
- **受纳腐熟水谷**
- **主通降**

### 小肠（受盛之官）
- **受盛化物**
- **泌别清浊**

### 大肠（传导之官）
- **传导糟粕**
- **吸收水分**

### 膀胱（州都之官）
- **贮尿和排尿**

### 三焦（决渎之官）
- **通行元气**
- **运行水液**

## 🌿 倪师讲解

### 脏腑与六经

倪海厦：**"脏腑是基础，六经是脏腑病变的外在表现。"**

- **太阳病** - 膀胱经病变
- **阳明病** - 胃、大肠经病变
- **少阳病** - 胆经病变
- **太阴病** - 脾经病变
- **少阴病** - 心、肾经病变
- **厥阴病** - 肝、心包经病变

### 脏腑与方剂

| 脏腑 | 代表方剂 | 应用 |
|------|----------|------|
| 心 | 炙甘草汤 | 心动悸、脉结代 |
| 肝 | 小柴胡汤 | 少阳证 |
| 脾 | 理中汤 | 太阴脾虚 |
| 肺 | 麻黄汤 | 太阳伤寒 |
| 肾 | 四逆汤 | 少阴寒化 |
