# 瘀血的概念

**所属分类：** 病因专题  
**倪海厦讲解来源：** 《伤寒论》、《金匮要略》

---

## 📖 基础定义

### 什么是瘀血？

**瘀血**是指血液运行不畅或停滞不行形成的病理产物，包括：
- 离经之血未能排出或消散
- 血液运行受阻，滞留于体内

**经典定义：** 王清任《医林改错》"元气既虚，必不能达于血管，血管无气，必停留而瘀"

---

## 🧬 瘀血的形成

### 病因病机

```
外伤 → 出血离经 → 瘀血内停
     ↓
气虚 → 推动无力 → 血行缓慢
     ↓
气滞 → 血行受阻 → 血液瘀滞
     ↓
寒凝 → 血液凝泣 → 运行不畅
     ↓
热灼 → 血液粘稠 → 瘀阻成块
     ↓
痰阻 → 脉络阻塞 → 血行不畅
```

### 关键因素
- **气** - 气为血之帅，气虚气滞皆可致瘀
- **寒** - 寒性凝滞，血遇寒则凝
- **热** - 热灼津血，血液粘稠
- **虚** - 正气不足，推动无力

## 🔍 瘀血证候

### 主要症状
| 部位 | 症状 |
|------|------|
| 疼痛 | 痛有定处，如针刺刀割 |
| 肿块 | 固定不移，触之硬满 |
| 出血 | 血色紫暗，夹有血块 |
| 紫绀 | 面色青暗，唇甲青紫 |
| 舌象 | 舌质紫暗，有瘀点瘀斑 |
| 脉象 | 脉涩或结代 |

### 常见部位表现
- **头部：** 头痛、眩晕、记忆力减退
- **胸部：** 胸痛、心悸、胸闷
- **腹部：** 腹痛、癥瘕积聚
- **四肢：** 麻木、疼痛、拘挛

## 💊 治法方药

### 活血化瘀法

| 证型 | 治法 | 代表方 |
|------|------|--------|
| 气滞血瘀 | 行气活血 | 血府逐瘀汤 |
| 气虚血瘀 | 益气活血 | 补阳还五汤 |
| 寒凝血瘀 | 温经活血 | 温经汤 |
| 热毒瘀结 | 清热化瘀 | 桃红四物汤合清热解毒药 |
| 痰瘀互结 | 化痰祛瘀 | 桂枝茯苓丸 |

### 倪师常用活血药

- 桃仁
- 红花
- 川芎
- 赤芍
- 莪术
- 三七
