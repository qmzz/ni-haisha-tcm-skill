# 神

**分类：** 精气神专题  
**来源：** 黄帝内经

---

## 📖 核心概念

神是人体生命活动、精神意识、思维情志的总体现。

> **《素问·移精变气论》：** "得神者昌，失神者亡。"

中医学中的“神”既指生命活力，也指精神意识活动，核心归于心，根本赖于精气充足。

---

## 🧠 神的层次

### 广义之神
- 人体一切生命活动的外在表现
- 包括神志、反应、面色、目光、言语、动作等

### 狭义之神
- 指精神、意识、思维、判断等心神活动
- 与心主神明关系最密切

---

## 🔄 神与精气血的关系

| 概念 | 关系 |
|------|------|
| **精** | 神的物质基础 |
| **气** | 神的动力来源 |
| **血** | 神的营养依托 |
| **心** | 神明所藏之处 |
| **肾** | 精充则神旺 |

### 基本规律
- **精足则神旺**
- **气足则神清**
- **血足则神安**
- **阴阳平衡则神守**

## 👁️ 临床诊神

| 状态 | 表现 | 临床意义 |
|------|------|----------|
| **得神** | 目光有神、反应灵敏、语言清楚 | 正气充足，病情较轻 |
| **少神** | 精神不振、倦怠懒言 | 气血不足，正气虚弱 |
| **失神** | 神志淡漠、目无光彩、反应迟钝 | 病重，脏腑精气衰败 |
| **假神** | 久病重病突然精神转好 | 危重转归，多非佳兆 |
| **神乱** | 狂躁、谵语、神昏、抑郁 | 痰火扰心、热入心包或情志内伤 |

## 🩺 常见神志病机

### 心神不宁
- 心悸、失眠、多梦、易惊
- 多见于心血不足、阴虚火旺

### 痰蒙心窍
- 神志模糊、胸闷、痴呆、昏迷
- 多见于痰浊内阻

### 热扰心神
- 烦躁不眠、谵语、狂躁
- 多见于实热、营血热盛

### 精亏神疲
- 健忘、迟钝、精神萎靡
- 多见于久病、肾精不足

## 🌿 倪师讲解

倪海厦：**"看病先看神。人有没有神，常常比他说什么更重要。神在，病还有转机；神散了，病就重了。"**

### 临床重点
1. **先看眼神** - 目光能反映心神存亡
2. **再看反应** - 反应快慢可见正气盛衰
3. **结合声音动作** - 神足则言行协调
4. **神与舌脉合参** - 不能只凭一个表现断神机
