# 太溪穴 (KI3)

**归经：** 足少阴肾经  
**定位：** 足内侧，内踝后方，当内踝尖与跟腱之间的凹陷处  
**特定穴：** 输穴、原穴

## 📍 穴位定位

### 标准定位
**《针灸大成》：** "在足内踝后，跟骨上动脉陷中。"

### 取穴方法
内踝尖与跟腱之间的凹陷处。

---

## 🩺 功能主治

### 功效
- **滋阴补肾** - 肾虚证
- **清热凉血** - 阴虚火旺
- **通调二便** - 二便不利

### 主治病证

**倪海厦：**
> "太溪为肾经输穴、原穴，主滋阴补肾，清热凉血。
> 凡肾虚、阴虚火旺，皆可取太溪。"

| 系统 | 病证 |
|------|------|
| **肾病** | 头痛、目眩、失眠、健忘、遗精、阳痿 |
| **咽喉病** | 咽干、咽痛 |
| **泌尿病** | 小便不利、尿频 |
| **妇科病** | 月经不调、带下 |
| **其他** | 齿痛、便秘 |

---

## 🔌 配伍应用

| 配伍穴位 | 功效 | 主治 |
|----------|------|------|
| **太溪 + 肾俞** | 滋阴补肾 | 肾虚 |
| **太溪 + 三阴交** | 调理月经 | 月经不调 |
| **太溪 + 照海** | 滋阴清热 | 阴虚火旺 |

- **艾条灸：** 10-15 分钟
