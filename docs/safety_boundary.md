# Safety Boundary

本项目定位为：

```text
中医学习、资料检索、辨证辅助、经方检索系统
```

不是：

```text
医疗诊断系统
自动开方系统
治疗建议系统
```

## 必须提示就医的红旗症状

内置红旗包括但不限于：

- 胸痛 / 胸闷 / 呼吸困难
- 意识不清 / 昏迷 / 抽搐
- 高热不退
- 咯血 / 便血 / 呕血
- 剧烈腹痛
- 中风相关表现
- 孕妇 / 婴儿 / 幼儿高风险场景

命中红旗后：

```text
停止输出方剂参考，提示优先联系专业医师或及时就医。
```

## Agent 使用纪律

- 不得输出“诊断结论”。
- 不得输出“照此服用/照此针刺”。
- 方剂、剂量、针灸内容只能作为资料参考。
- 所有医学相关结果应带免责声明和来源状态。
