## 🔧 核心模块说明

### DiagnosisEngine（诊断引擎）
- **功能**：基于六经辨证和八纲辨证的辨证辅助
- **覆盖**：50+ 方剂（伤寒论 + 金匮要略）
- **能力**：症状分析、证型判断、方剂推荐、加减建议
- **使用**：`engine.analyze(["发热", "恶寒", "无汗"])`

### FormulaRecommender（方剂推荐器）
- **功能**：基于伤寒论和金匮要略的方剂匹配
- **覆盖**：72 首核心方剂
- **能力**：症状-方剂匹配、加减推荐
- **使用**：`recommender.recommend(["发热", "汗出"])`

## 📊 数据文件

### symptom_formula.json
- 55 个常见症状的方剂映射
- 支持辨证辅助引擎的症状-方剂匹配
- 每个症状关联 1-4 个方剂

### concept_graph.json
- 100+ 中医概念节点
- 200+ 概念关系边
- 涵盖阴阳五行、脏腑经络、六经辨证、方剂穴位、症状关系

### learning_paths.json
- 4 个学习阶段（入门/基础/进阶/专业）
- 每阶段包含 3-4 个模块
- 支持渐进式学习路径

## 🚀 使用方式

### CLI 使用
```bash
python3 cli.py diagnose "发热,恶寒,无汗,脉浮紧"  # 辨证辅助
python3 cli.py formula 桂枝汤                    # 查询方剂
python3 cli.py herb 麻黄                         # 查询药材
python3 cli.py acupoint 合谷                     # 查询穴位
python3 cli.py stats                             # 统计信息
```

### Python API
```python
from internal.diagnosis_engine import DiagnosisEngine

engine = DiagnosisEngine()
result = engine.analyze(["发热", "恶寒", "无汗", "脉浮紧"])
print(result)
```
