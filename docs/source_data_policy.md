# 原始数据使用原则

本项目知识来源以 `project/nihaixia/` 下从 PDF 提取的结构化 JSON 文件为准。

## 硬性原则

1. **尊重原始数据**：不得凭模型记忆或主观理解扩写药材、方剂、医案、剂量、针灸操作等内容。
2. **可追溯**：新增或补全知识条目时，应记录来源文件、页码或原文片段位置。
3. **不乱编**：无法在原始 JSON 或既有知识文件中找到依据时，只能标记为待考/待补充，不得补成确定结论。
4. **安全优先**：系统定位为中医理论学习、资料检索和辨证思路参考，不替代医师诊断和治疗。
5. **结构优先于扩写**：优先建设索引、校验、引用、测试和安全边界，再做内容补全。
